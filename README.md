## 🚀 프로젝트 실행 방법

database.properties 올바른 데이터베이스 정보를 입력해주세요. </br>
docs 폴더에 data.sql 쿼리문을 실행해주세요.

## 🐳 Docker로 실행하기

root 디렉토리에서 다음 명령어를 실행해주세요.

```bash
docker-compose up
```

코드가 변경되었을떄 최신 이미지를 받아오고 컨테이너재시작하려면 다음 명령어를 실행해주세요.

```bash
docker-compose up --build
```

컨테이너를 종료하려면 다음 명령어를 실행해주세요.

```bash
docker-compose down
```
